# RAG & NLP Project

This folder contains project demonstrating Retrieval-Augmented Generation (RAG), Natural Language Processing (NLP), Large Language Models (LLMs), and Generative AI applications developed in Python.

## Technologies

- Python
- LangChain
- OpenAI GPT-4o-mini
- OpenAI Embeddings
- FAISS
- Retrieval-Augmented Generation (RAG)
- Natural Language Processing (NLP)
- Generative AI

## Project

### NLP RAG Assistant

Built a Retrieval-Augmented Generation (RAG) pipeline in Python using LangChain, OpenAI GPT-4o-mini, FAISS, and OpenAI embeddings, applying NLP and Generative AI techniques for semantic search and question-answering over regulatory document.
